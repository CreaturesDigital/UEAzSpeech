# Unreal Engine Plugin: AzSpeech

![image](https://user-images.githubusercontent.com/77353979/201452810-0e92639b-bc72-4060-8271-a6333edaf797.png)

A Unreal Engine plugin that integrates Azure Speech Cognitive Services to the Engine by adding functions to perform recognition and synthesis via asynchronous tasks.
